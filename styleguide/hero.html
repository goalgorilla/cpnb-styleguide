Hero default

The hero is the first section on each detail page (so not present in overviews).

  • It has a large background image. Images should be in greyscale.
  • It has a large title, which should always be the only H1 tag of the page
  • It can optionally have extra info, depending on the page type.

Hero campaign version

The campaign version of the hero has 3 more elements:

  • The campaign logo, subtle in greyscale and has a hover effect. Hidden on mobile.
  • Buttons to let the user take action. We see here the LU version. The AN version has only one (.button-view) button.
  • The status of the project. Either Live or Archived.
